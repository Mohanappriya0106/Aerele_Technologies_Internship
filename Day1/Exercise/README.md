# Day 1 - Python Exercise

## Objective

This exercise demonstrates the Python concepts learned during Day 1 of the Internship.

## Concepts Covered

- Virtual Environment (`venv`)
- Installing packages using `pip`
- Context Managers (`with`)
- List Comprehensions
- Pythonic Coding Style
- Java-style vs Pythonic code comparison

## How to Run

### 1. Create Virtual Environment

```bash
python3 -m venv venv
```

### 2. Activate

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run

```bash
python main.py
```

```bash
python pythonic_comparison.py
```

## Learning Outcome

This exercise demonstrates how to:

- Create and use a virtual environment
- Install third-party Python packages
- Use context managers for safe file handling
- Use list comprehensions for Pythonic code
- Compare Java-style and Pythonic programming approaches
